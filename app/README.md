# 🚀 FastAPI DevOps Project

## 📌 Overview

This project demonstrates a complete DevOps pipeline including Docker, Jenkins CI/CD, and monitoring using Prometheus & Grafana.

## 🐳 Tech Stack

- FastAPI
- Docker & Docker Compose
- Jenkins (CI/CD)
- Prometheus (Monitoring)
- Grafana (Visualization)

## ⚙️ Features

- REST API with FastAPI
- Persistent storage using volumes
- Automated CI/CD pipeline
- Real-time monitoring dashboard

## 🚀 Run Project

```bash
docker-compose up -d
```
